Proving the Intersection of Sets is Empty Example

Let A and B be sets, and let C be a non-empty set. Prove that

A∩B=∅⟺(A×C)∩(B×C)=∅.
Solution

In order to prove the biconditional, we first prove sufficiency, then necessity.

  • (⇒): Let A∩B=∅. Assume for contradiction that (A×C)∩(B×C)=∅. Then, there exists x in (A×C)∩(B×C). Since C=∅ and by definition of set intersection, we further observe that x=(u,v) for some u∈A∩B and v∈C. The fact that u∈A∩B is a contradiction to the assumption that A∩B=∅. Therefore, (A×C)∩(B×C)=∅ by contradiction.

  • (⇐): Let (A×C)∩(B×C)=∅. Assume for contradiction that A∩B=∅. Then, there exists x in A∩B. Further, since C=∅ by assumption, then there exists c in C. Thus, (x,c) is an element of (A∩B)×C. We also observe that

    (A∩B)×C=(A×C)∩(B×C),

    hence (x,c)∈(A×C)∩(B×C) which is a contradiction to the assumption that (A×C)∩(B×C)=∅. Therefore, A∩B=∅ by contradiction.
